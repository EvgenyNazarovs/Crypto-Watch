# Crypto-Watch

A straightforward cryptocurrency tracker for Top 20 Coins, made with Vue JS and CoinGecko API.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
